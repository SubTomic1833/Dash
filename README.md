# COSMO-RUN

A fast-paced, minimalist endless runner built with vanilla JavaScript and HTML5 Canvas. Jump over spikes, land on blocks for bonus points, and see how high you can push your score as the game speeds up.

Inspired by Geometry Dash

## Play

Link : https://subtomic1833.github.io/COSMO-RUN/

## Controls

- Use the **SPACEBAR** to jump.

## Rules

- Survive as long as you can while obstacles speed up over time.
- **Spikes** — must be cleared with a jump. Touching one ends the run.
- **Blocks** — can be jumped on top of for **+50 bonus points**, or cleared entirely. Running into the side (or under-jumping) ends the run.
- Score increases automatically over time, plus bonus points from block landings.
- Every 600 points, you level up — obstacles get faster and spawn more frequently.
